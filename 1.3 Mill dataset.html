How to merge CSV files and show that Python can open larger files than Excel¶

Make sure all of the files you want to merge are in the root folder together or at least the same folder as where your Python code is running. Mine is the root.

Excel can only read about one million rows, whereas, Python has a much larger threshold. Here I will show you the difference. I have three files that I will merge to create a 1.3 million record dataset.

Note that the merging order is based on the file name order. A-Z

What happens in Excel¶

Here is what happens when you open a file that is too big for Excel. You got this pop-up and you can see the bottom of Excel.
